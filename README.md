# 📊 E-commerce Sales Data Analysis

## 📌 Project Overview

This project analyzes an e-commerce sales dataset to uncover insights using Python.

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib
* Seaborn

## 📂 Dataset

The dataset contains information about:

* Orders
* Customers
* Products
* Sales

## 🔍 Key Analysis Performed

* Data Cleaning
* Category-wise Sales Analysis
* Monthly Sales Trend
* Region-wise Analysis
* Top Selling Products

## 📈 Visualizations

* Sales by Category
* Monthly Sales Trend
* Top Products Chart

## 🧠 Key Insights

* Technology category has the highest sales
* Sales vary across months (peak observed in specific months)
* Certain products contribute significantly to revenue

## 🚀 Conclusion

This project demonstrates basic data analysis skills including data cleaning, grouping, and visualization.
